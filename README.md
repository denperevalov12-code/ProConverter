# ProConverter — это визуально эффектный онлайн-конвертер изображений с поддержкой 13 форматов (JPEG, PNG, WEBP, TIFF, RAW, GIF, SVG, HEIC, BMP, PSD, PDF, ICO, AVIF).

# Основные возможности:

Drag & Drop — перетащите файл в зону загрузки или выберите через файловый менеджер

Выбор направления конвертации — укажите исходный формат (или автоопределение) и целевой формат

Регулировка качества — для lossy-форматов (JPEG, WEBP, HEIC, AVIF) доступен ползунок 10-100%

Предпросмотр — до и после конвертации с метаданными (разрешение, размер)

Скачивание результата — одним кликом сохраняете сконвертированное изображение

# Ограничения:

Некоторые форматы (RAW, PSD, PDF, TIFF) не могут быть декодированы браузером — появится предупреждение

Кодирование в GIF даёт статичное изображение, не анимированное

HEIC работает только в Safari (iOS/macOS)

Не все форматы можно создать (RAW, PSD, PDF, TIFF, HEIC — только как исходные)
